# BITECK ETHAN Formation

Application React + Vite avec backend Express et intégration Supabase.
Pour les differentes formation en presentiel que je vais donner dispenser par C&B Services

## Lancer le projet

**Prérequis :** Node.js

1. Installer les dépendances :
   `npm install`
2. Configurer les variables d’environnement nécessaires pour Supabase et le serveur.
3. Démarrer en local :
   `npm run dev`

## Commandes utiles

- `npm run dev`
- `npm run lint`
- `npm run build`
